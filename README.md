## Technologies used to build this App.

- Frontend: React, JavaScript.
- Backend: NodeJS, ExpressJS, Sequelize
- Designs: CSS
- Database: PostgreSQL

## What you can do.
 Search for a country, sort them from A-Z, Z-A, by Population or Filter by Region or Activity Season.
